### 標頭檔：stdio.h

### 函式：int getchar (void)

### glibc 手冊函式說明：
The getchar function is equivalent to getc with stdin as the value of the stream argument.

### 輸入
The getchar function is equivalent to getc with stdin as the value of the stream argument.

與**`getc`**相同，唯一差別在**`getchar`**是以 stdin 為 stream argument。

### 回傳值
如同**`getc`**
